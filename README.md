**Description**: This application is a straightforward Point of Sale (POS) system for a small retail establishment that is menu-driven.



**Cashiers are able to**: 

* Put things in cart and take them out
* Examine the product catalogue and cart
* After checking out, apply the 10% tax and look for any discounts (5% if the total exceeds $5000).
* Receiver a formatted receipt
* Run multiple transactions at once



**HOW TO RUN**:



1. Save the application as pos\_system.py
2. Execute it in a Python environment such as IDLE
3. Select selections from the on-screen menu (choices 0-6)



**Limitations and Assumptions**



* Case sensitivity applies to product names
* Only cash-like payments (no credit card information is accepted). Currency is Jamaican Dollars (JMD)
* Data is lost when the program ends; there is no database or file persistence
* The discount (5% if total <$5000) is only applied once during checkout.
